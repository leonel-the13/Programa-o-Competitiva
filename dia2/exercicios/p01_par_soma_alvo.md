# Problema 1: Par com Soma Alvo

**Dificuldade**: Médio  
**Tempo**: 1 segundo

## Descrição

Dado um array ordenado e um valor target, encontre dois números cuja soma seja igual a target.

## Entrada

Primeira linha: N, target (2 ≤ N ≤ 10^5, -10^9 ≤ target ≤ 10^9)  
Segunda linha: N inteiros ordenados (-10^9 ≤ aᵢ ≤ 10^9)

## Saída

Imprima dois números ou "-1" se não existir.

## Exemplo

**Entrada**:

```
5 9
2 7 11 15 27
```

**Saída**:

```
2 7
```

## Dica

Use two pointers: um no início, outro no fim.
