## 🧠 REGRAS DE OURO

1. ⏱️ **Máximo 45 min por problema**
   - Se travar → veja a solução → entenda → implemente sozinho

2. 📝 **Mantenha caderno de padrões**
   - "Esse é two pointers"
   - "Esse é BFS em grid"
   - "Esse é knapsack"

3. 🎯 **Reconhecimento de padrão = vitória**
   - Maratona é sobre RECONHECER rapidamente

4. ⚡ **Não resolva só fáceis**
   - Difíceis te preparam para pensar sob pressão

5. 🔄 **Refaça problemas que errou**
   - Erro não revisado = padrão não aprendido
